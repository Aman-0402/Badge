## Quick Start — Earn 4 Badges in One Day

These four can be earned within hours, no collaborator needed:

| Step | Badge | Time |
|------|-------|------|
| 1 | **Quickdraw** — open an issue, close it within 5 min | 2 min |
| 2 | **YOLO** — create branch, open PR, merge without review | 5 min |
| 3 | **Pair Extraordinaire** — co-author a commit with a friend | 10 min |
| 4 | **Galaxy Brain** — answer a Q&A discussion, get it marked | 15 min |
