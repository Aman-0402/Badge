# Contributing to Badge

Thanks for your interest in contributing! This repo tracks GitHub Achievement badges — contributions that help document, improve, or expand the tracker are welcome.

---

## Ways to Contribute

- Add missing badge information
- Improve existing badge descriptions or tips
- Fix typos or broken links in docs
- Share your own achievement strategy in a Discussion
- Help others earn badges by co-authoring commits

---

## Getting Started

1. **Fork** this repository
2. **Clone** your fork:
   ```bash
   git clone https://github.com/<your-username>/Badge.git
   cd Badge
   ```
3. **Create a branch:**
   ```bash
   git checkout -b your-branch-name
   ```
4. **Make your changes**
5. **Commit** with a clear message:
   ```bash
   git commit -m "docs: describe what you changed"
   ```
6. **Push** and open a Pull Request

---

## Pull Request Guidelines

- Keep PRs focused — one change per PR
- Reference any related issue in the PR description (e.g. `Closes #5`)
- PR title format: `type: short description`
  - `docs:` for documentation changes
  - `fix:` for corrections
  - `feat:` for new content

---

## Good First Contributions

Look for issues labeled **good first issue** — these are small, well-defined tasks perfect for first-time contributors.

---

## Co-authoring (Pair Extraordinaire badge)

Want to earn the **Pair Extraordinaire** badge? Open a PR with a co-authored commit:

```
Co-authored-by: Your Name <your-email@example.com>
```
