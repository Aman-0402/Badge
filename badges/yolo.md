# YOLO

## What It Is
Awarded for merging a pull request without requesting or receiving any code review. One-time badge, no tiers.

## How to Earn It

1. Create a branch in any repo you own
2. Make any change and push it
3. Open a pull request
4. **Do not add any reviewers**
5. Merge it yourself immediately

That's it. No waiting, no approvals needed.

## Step by Step

```bash
# Step 1 — create branch
git checkout -b yolo-test

# Step 2 — make any small change
echo "test" >> README.md

# Step 3 — commit and push
git add README.md
git commit -m "test: yolo commit"
git push origin yolo-test
```

Then on GitHub:
- Open the PR from `yolo-test` → `main`
- Leave Reviewers section empty
- Click **Merge pull request**

## Important

- Must be a **merged** PR, not just opened
- You must be the one merging (owner or collaborator with write access)
- The badge appears on your profile within a few hours
