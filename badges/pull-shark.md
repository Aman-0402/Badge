# Pull Shark

## What It Is
Awarded for getting pull requests merged. One of the most common and progressive badges — it has four tiers.

## Tiers

| Tier | Merged PRs Required |
|------|-------------------|
| Bronze | 2 |
| Silver | 16 |
| Gold | 128 |
| Platinum | 1024 |

## How to Earn It

1. Create a branch in any public or private repo
2. Make a change (even a small doc fix counts)
3. Open a pull request
4. Get it merged

Merging your own PRs on your own repos counts. You do not need someone else to merge it.

## Tips

- Every merged PR across all your repos counts — not just one repo
- Contributing to open source is the fastest way to stack PRs
- Doc fixes, typo corrections, and README improvements are easy PRs to get merged
- Repos to find beginner tasks: [goodfirstissue.dev](https://goodfirstissue.dev), [up-for-grabs.net](https://up-for-grabs.net)

## What Does NOT Count

- Closed PRs (not merged)
- Draft PRs
- PRs that were reverted immediately (edge case)
