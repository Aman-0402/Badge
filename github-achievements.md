## Action Plan (Easiest Wins First)

### 1. Pair Extraordinaire (Fast)
- Co-author a commit in any merged PR
- Add this trailer to a commit message:
  ```
  Co-authored-by: Name <email@example.com>
  ```
- Works on public OR private repos if the PR gets merged

### 2. YOLO (Fast)
- Open a PR on your own repo and merge it yourself without requesting review

### 3. Quickdraw (Fast)
- Open an issue or PR, then close it within 5 minutes

### 4. Pull Shark (Ongoing)
- Keep contributing PRs to open-source projects
- Every merged PR counts toward tiers
- CertiVerse and other active projects = easy source

### 5. Galaxy Brain (Requires community)
- Find GitHub Discussions on popular repos
- Answer questions thoroughly — someone (or the maintainer) must mark it as "Answered"

### 6. Public Sponsor (Costs money)
- Go to GitHub Sponsors
- Pick any developer/org you use and sponsor even $1/month

### 7. Starstruck (Long game)
- Build something useful and share it widely
- Twitter, Reddit, dev communities
